Nav Styles

Styles can be set for nav-item-standard and nav-item-gradient components with the following @Inputs

  • [itemname] - string, name of nav item
  • [itemstyle] - string, shape of item - rectangle or parallelogram
  • [colorfadespeed] - string, fade speed of color (text color) - vslow, slow, medium, fast, or vfast
  • [fadetocolortext] - string, color to fade text to - css rgb, hex, or color name string
  • [smooth] - use smooth border-radius or not - true or false
  • [fadespeed]* - string, fade speed of background-color - vslow, slow, medium, fast, or vfast

* only for nav-item-gradient
