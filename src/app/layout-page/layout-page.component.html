Layout

A set of CSS layout classes to define widths floats and spacing. Layout styles can be found here.

Layout Styles

Layouts are based on 30 equal columns so that many different layouts can be accommodated, these can be used differently for 5 different screen sizes. A few other classes such as "*-qu" for quarter widths, "off-*" for offsets, and some float classes can be used for more precise positioning. These styles are built using flexbox and can be adjusted and changed using the same approach.

Some useful class definitions

  • .container - container for layout elements, default width is 80% with 10% margin on each side
  • .row - container for layout elements (use row within container and element widths within row)
  • .fl-left - float row elements to the left (or right with .fl-right) of the container (can also just be done more precisely with offsets)
  • .g-1 - general width class (g) for when you just want the same width (here 1 column) across all screens
  • .xs-5 - define element with width of 5 columns (1/6th) of the container element for extra small screen sizes
  • .m-qu - define element with width of 1 quarter the size of the container element for medium screen sizes
  • .off-l-4 - define offset width of element (4) for screen size (l can be used with all sizes including g)
