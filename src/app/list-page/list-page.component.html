List

A collection of CSS and Javascript list and list effect designs.

List Styles

Styles can be set for list-item components with the following @Inputs

  • [itemname] - string, name of list item
  • [itemstyle] - string, 'list-item' and any other classes you wish to apply, here I have used 'b1-primary'
  • [hoverable] - boolean, if hoverable the sublist expands when hovered, otherwise a button is provided using the button style input
  • [buttonStyle] - string, triangle or circle, triangle rotates when clicked, circle goes from filled when unclicked to empty when clicked
  • [isPartial]* - boolean, if true shows the edges of the subitems below each item in the list
  • [subliststyle]* - string, list-subitems or list-partial-subitems if isPartial is true
  • [subitems] - array of subitems. Each subitem is simply a string to hold the subitem name
  • [subitemstyle] - string, 'list-subitem' and classes for styles of subitems, here I have used 'b1-neutral' and 'margin-moveable-opacity-medium' for smooth movement

*These should match the style in the containing ul tag

e.g. if the ul class is "list-hideable" isPartial should be false and the subliststyle should be "list-subitems"

If the ul class is "list-partial-hideable" isPartial should be true and the subliststyle should be "list-partial-subitems"
